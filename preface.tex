\chapter*{Introduction}
\addcontentsline{toc}{chapter}{Introduction}

Superadditive set functions offer a way to express the relationship between subsets of some finite ground set.
This is used in countless fields, e.g., for modeling the utility of a player based on their received items in combinatorial auctions, or for modeling the value of coalitions in cooperative game theory.

However, in applying set functions to a real-world problem, one quickly reaches a significant roadblock: their size.
A set function assigns a value to every subset of a set $ N $.
It is thus easy to see that the size of a set function grows exponentially in the size of $ N $.
In many applications, getting even a single value of the set function is hard -- it costs, e.g., money, time, computational resources.

In this thesis, we investigate a different approach.
We work with a \emph{budget} of how many subsets we are willing to reveal at maximum.
We assume the function comes from a given class of set functions, and we can thus bound the remaining values.
This introduces \emph{ambiguity} to our model.
Our aim is to choose which subsets to reveal, such that the resulting ambiguity in the values of the unknown subsets is minimized.
We form this as an optimization problem, and explore multiple approaches for solving it.
First, we investigate simple search techniques, and later we turn to reinforcement learning algorithms.

\section*{Organization and Contributions}

We begin by introducing set functions, their basic properties and classification.
On top of this, we build the theory of cooperative games.
Further, we offer an introduction of reinforcement learning, focusing mainly on the PPO algorithm.
We continue with defining incomplete set functions, formally introducing ambiguity to our model.
To measure the ambiguity, we introduce the \emph{divergence}.
Finally, we define the online and offline Principal's problems, which aim to minimize the divergence in an online and offline setting, respectively.

This setting was originally stated by \cite{uradnik2024reducing}, while studying this setting purely in cooperative game theory.
In this thesis, we extend these ideas to be applicable to general set functions.
We offer a more extensive theoretical analysis of the approaches proposed by \cite{uradnik2024reducing}, including comparison of the online and offline approaches, and a formal proof of the time complexity of all proposed algorithms.
The divergence is a generalization of their utopian gap, which has a motivation purely in cooperative game theory, to a more general valuation applicable to general set functions.
We analyze this generalization theoretically, and then offer an empirical evaluation and comparison of the valuations.
