\chapter*{Introduction}
\addcontentsline{toc}{chapter}{Introduction}

Superadditive set functions offer a way to express the relationship between subsets of some finite ground set.
This is used in countless fields, e.g., for modeling the utility of a player based on their received items in combinatorial auctions, or for modeling the value of coalitions in cooperative game theory.

However, in applying set functions to a real-world problem, one quickly reaches a significant roadblock: their size.
A set function assigns a value to every subset of a set $ N $.
It is thus easy to see that the size of a set function grows exponentially in the size of $ N $.
\filip{for the memes:}
If $ N $ contains more than 300 elements, the number of subsets, and thus the size of the set function, is larger than the number of particles in the universe.

Throughout the literature, we can find many ways to circumvent this problem.
Most commonly, this is done by restricting the allowed structure of the set function, and based on that creating an oracle which can compute the value of a subset from a smaller amount of information.
For example, consider a set function $ f $ which is additive, meaning $ \fce{f}{S} = \sum_{i \in S}^{} \fce{f}{\left\{ i \right\}} $.
Then the whole set function is precisely given by just the values $ \fce{f}{\left\{ i \right\}} $ for all $ i \in N $.
While this solution is sufficient in many situations, it vastly limits the set functions we are allowed to work with.

In this thesis, we investigate a different approach.
We assume that, while the full set function is unknown to us, getting even a single value of it is hard -- it costs us, e.g., money, time, resources.
We work with a \emph{budget} of how many subsets we are willing to reveal at maximum.
Our aim is to choose which subsets to reveal, such that the values of the \emph{ambiguity} in the values of the unknown subsets is minimized.


\section*{Organization and Contributions}

\cite{uradnik2024reducing} have proposed a framework for solving this problem, which is built on top of cooperative game theory.
In this thesis, we extend their ideas.
We begin with an introduction to set functions, including basic properties and their classification.
We then formally define related concepts in cooperative game theory, which we later use when comparing our results to \cite{uradnik2024reducing}.
Further, we introduce reinforcement learning, which we employ to approximate online learning.

We continue with defining incomplete set functions, formally introducing ambiguity to our model.
To measure the ambiguity, we introduce the \emph{divergence} as a generalization of the utopian gap \citep[][Definice xx]{uradnik2024reducing}.
Finally, we define the online and offline Principal's problems, which aim to minimize the divergence in an online and offline setting, respectively.

We continue to our main contributions.
We offer a more extensive theoretical analysis of the approaches proposed in \cite{uradnik2024reducing}, including comparison of the online and offline approaches, and a formal proof of the time complexity of all proposed algorithms.
We generalize their utopian gap, which has a motivation purely in cooperative game theory, to a more general valuation applicable to general set functions.
We analyze this generalization theoretically, and then offer an empirical evaluation and comparison of the valuations.
