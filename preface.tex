Superadditive\david{proč je tady superadditive? Nechceš za tuhle větu dát ještě větu o superadditivních?} set functions offer a way to express the relationship between subsets of some finite ground set.
This is used in countless fields, e.g., for modeling the utility of a player based on their received items in combinatorial auctions, or for modeling the value of coalitions in cooperative game theory.
\david{ještě bych vymyslel další příklady}

However, in applying set functions to a real-world problem, one quickly reaches a significant roadblock: their size.
A set function assigns a value to every subset of a set $ N $.
It is thus easy to see\david{vykašlal bych se na to co je easy to see, to je subjektivní. Tohle je prostě fakt} that the size of a set function grows exponentially in the size of $ N $.
In many applications, getting\david{obtaining? Ale getting je taky fajn, jen trochu víc general} even a single value of the set function is hard -- it costs, e.g., money, time, \david{or?} computational resources.

In this thesis, we investigate a different approach.\filip{tady ted neni jasny z ceho "different", protoze se jeste rozhoduju jestli nad tohle nedam jeden odstaec o ostatnich pristupech}\david{jn souhlas, ten odstavec by to podle mě chtělo}
We work with a \emph{budget} of how many subsets we are willing to reveal at maximum.\david{tahle věta zní divně. At maximum mi zní nic moc. Celkově bych řekl něco jakože uvažujeme, že dokážeme získat pouze omezený počet subsetů? Ten emphasis na budget mi evokuje, že je to definice toho slova. A jako jo, ale až na skoro konci bakalářky}
We assume the function comes from a given class of set functions, and we can thus bound the remaining values.\david{tohle je taky trochu random - nemělo by se tímhle začít?}
This introduces \emph{ambiguity} to our model.
Our aim is to choose which subsets to reveal, such that the resulting ambiguity in the values of the unknown subsets is minimized.
We form\david{frame?} this as an optimization problem, and explore multiple approaches for \david{(approximately)} solving it.
First, we investigate simple search techniques, and later we turn to reinforcement learning algorithms\david{RL je taky search, takže tímhle chceš říct, že není simple? :D}.

\section*{Organization and Contributions}

We begin by introducing set functions, their basic properties and classification\david{jen classes?}.
On top of this\david{the set functions}, we build the theory of cooperative games.
Further, we offer an introduction of\david{to} reinforcement learning, focusing mainly on the\david{a algorithm called PPO, which is a state-of-the-art RL algorithm?} PPO algorithm.
We continue with\david{by?} defining incomplete set functions, formally introducing ambiguity to our model\david{není jasné co vůbec je our model}.
To measure the ambiguity, we introduce the \emph{divergence}.\david{Hmm, není torchu divný první říct, "formally introducing ambiguity to our model" a až pak říct že jí měříme divergencí?}
Finally, we define the online and offline Principal's problems\david{paní principálovou bych z toho vynechal, působí jakože spadla z nebe :)}, which aim to minimize the divergence in an online and offline setting, respectively.\david{tohle by se dalo říct víc intuitivně...}

This setting was originally stated by\david{in?} \cite{uradnik2024reducing}, while\david{where the authors focus only on the GT setting} studying this setting purely in cooperative game theory.
In this thesis, we extend these ideas to be applicable to general set functions.
We offer a more extensive theoretical analysis of the approaches proposed by \cite{uradnik2024reducing}, including comparison of the online and offline approaches, and a formal proof of the time complexity of all proposed algorithms.
The divergence is a generalization of their utopian gap, which has a motivation purely\david{purely je za mě navíc} in cooperative game theory, to a more general valuation applicable to general set functions.
We analyze this generalization theoretically, and then offer an empirical evaluation and comparison of the valuations.
