\chapter{Problem Definition}

We will now define the model we will be working in.
Suppose we have a real-world situation, which can be modelled using a set function over some set $ N $.
The most natural example of this is cooperative game theory, where we have a set of players $ N $, who are, say, starting a company together.
Now the players want to split the earnings in a fair way, using the Shapley value.
However, as there are many of them, the characteristic function $ v $ is huge, requiring the knowledge of values for every coalition $ S \subseteq N $.

Without the knowledge of all the values in $ v $, they will not be able to compute the Shapley value.
This gives rise to \emph{ambiguity} in the distribution of earnings -- a player may claim he's more valuable to the grand coalition than he actually is, requiring a bigger share of the profit.

In \cite{uradnik2024reducing}, we have proposed a framework where the players select a portion of coalitions for which to reveal the values, such that the ambiguity is minimized.
In this thesis, we are extending this framework, both in theoretical and practical aspects, most notably by introducing the more general set functions and regret minimization. % TODO: možná regret ne xD

\section{Incomplete Set Functions}

First, let us formalize the set functions with incomplete information.

\begin{defi}[Incomplete set function]
  Let $ f: \pot N \to \R $ be a set function.
  An \emph{incomplete set function} is $ \left( f, \k \right) $, where $ \k \subseteq \pot N $ are the \emph{known values} of the set function.
\end{defi}

% TODO: jak se liší od partial set funcs
Notice that the $ f $ in the pair $ \left( f, \k \right) $ is still the \emph{complete} set function.
We are just restricting ourselves to using only the values from $ \k $ in computations.
This may seem unnatural at first, but it will help us in subsequent definitions.

\begin{defi}[$ \C $-extension]
  Let $ \C $ be a class of set functions, and let $ \left( f, \k \right) $ be an incomplete set function, such that $ f \in \C $.
  Then $ g \in \C $ is a \emph{$ \C $-extension of $ \left( f,\k \right) $}, if \[
    \left( \forall S \in \k \right)\quad \fce{f}{S} = \fce{g}{S}.
  \]
  The class of all $ \C $-extensions of $ \left( f,\k \right) $ is denoted by $ \fce{\C}{f, \k} $.
\end{defi}
% TODO: tady možná říct že tohle už asi někdy někoho napadlo

By assuming that $ f \in \C $, the $ \C $-extensions are a class containing every viable candidate for $ f $ -- it doesn't contradict anything we know about $ f $.
The set $ \fce{\C}{f, \k} $ is non-empty, as $ f $ is always a $ \C $-extension of itself, by the assumption of $ f \in \C $.
Further, if $ \k = \pot N $, then $ \fce{\C}{f, \k} $ is of size 1 -- only containing $ f $ itself.
Note that the set of $ \C $-extensions can have size 1 even for $ \k \neq \pot N $, as the following example shows.

\begin{example}[ ]
  Let $ f: \pot N \to \R $ be an additive set function, and $ \left( f, \k \right) $ be an incomplete set function, where $ \k = \left\{ \left\{ i \right\} \suchthat i \in N \right\} $.
  Then $ \fce{\cAdditive}{f, \k} $ is of size 1.
\end{example}

Now that we have all the candidates, we would like to bound the possible values of $ f $ for $ S \notin \k $.
As we will mostly focus on superadditive extensions, we will first define so-called \emph{minimal information} which is required to be known in order for all the bounds on the other values to be well defined.

\begin{defi}[Minimal information]
  An incomplete set function $ \left( f,\k \right) $ \emph{has minimal information}, if $ \k_0 \subseteq \k $, where \[
    \k_0 \deq \left\{ \emptyset, N \right\} \cup \left\{ \left\{ i \right\} \suchthat i \in N \right\}.
  \]
\end{defi}

% od teď v(0) = 0 & proč to nevadí
From this point on, we will limit ourselves to incomplete set functions with minimal information, where further $ \fce{f}{\emptyset} = 0 $.
This restricts the incomplete set functions we are working with, but only by a small margin, as we require only a \emph{linear} number of values to be known, compared to the exponential number of all values, which are still potentially unknown.
On the other hand, this restriction enables us to \emph{bound} the unknown values of $ f $, assuming that $ f $ is superadditive.

% upper a lower bound
\begin{defi}[Lower/upper function]
  Let $ f $ be a superadditive set function, and $ \left( f, \k \right) $ be an incomplete set function with minimal information.
  Then the \emph{lower function} $ \lf f $ is defined as \[
    \fce{\lf f}{S} \deq \max_{\substack{S_1, \dots, S_k \in \k \\ \bigcup_i S_i = S \\ S_i \cap S_j = \emptyset}} \sum_{i=1}^k \fce{f}{S_i},
  \]
  and the \emph{upper function} $ \uf f $ is defined as \[
    \fce{\uf f}{S} \deq \min_{\substack{T \in \k \\ T \subseteq S}} \fce{f}{T} - \fce{\lf f}{S \setminus T}.
  \]
\end{defi}

\begin{prop}[ ]
  Let $ f $ be a superadditive set function, and $ \left( f, \k \right) $ be an incomplete set function with minimal information.
  Then \[
    \left( \forall S \subseteq N \right)\quad \fce{\lf f}{S} \leq \fce{f}{S} \leq \fce{\uf f}{S}.
  \]
\end{prop}
% TODO: proof of this?

Further, as the following proposition shows, these bounds are tight.

\begin{prop}[ ]
  Let $ f $ be a superadditive set function, and $ \left( f, \k \right) $ be an incomplete set function with minimal information.
  Then for $ S \in \pot N \setminus \k $, there exist $ g,h \in \fce \cSuperadditive{f, \k} $, such that \[
    \fce{g}{S} = \fce{\lf f}{S}, \qquad \fce{h}{S} = \fce{\uf f}{S}.
  \]
\end{prop}

We would now like to measure the ambiguity of the incomplete set function.
We will do that by measuring the size of the set of $ \cSuperadditive $-extensions of $ \left( f, \k \right) $.
% \david{Možná říct, že buď máme jeden bod nebo kontinuum, ale různé dimenze. Takže nejde použít něco jako objem, ale musíme dělat tenhle divnej trik s normou / Shapleyho hodnotou?}
Intuitively, we would like to quantify the \emph{volume} of the set of $ \cSuperadditive $-extensions.
However, this is not possible, as the dimension of the set changes based on the size of $ \k$.
This leads us to measuring the distance between the upper and lower functions, using an arbitrary function $ \ell $.
The most obvious choice of $ \ell $ is some norm, as norms are well studied, well understood and have nice properties.
However, we state the following definition for an arbitrary $ \ell $, and we will see an example of $ \ell $ which is not a norm, but which gives a nice interpretation to the \textquote{size} of the $ \cSuperadditive $-extensions based on the context in which it is used.

\begin{defi}[Divergence]
  Let $ f $ be a superadditive set function, and $ \left( f, \k \right) $ be an incomplete set function with minimal information.
  Further, let $ \ell: \funcs \times \funcs \to \R^+_0 $.
  Then the \emph{set function divergence} of $ \left( f, \k \right) $ is $ \Delta_f^{\ell}: \pot{\pot N \setminus \k_0} \to \R^+_0 $, given by \[
    \divg \ell \k \deq \fce{\ell}{\lf[\k \cup \k_0] f, \uf[\k \cup \k_0] f}.
  \]
\end{defi}
% TODO: $ \ell $ pak nazýváme "divergence function"

Notice that the divergence is defined on the power set of the set of unknown values, without the minimal information $ \k_0 $.
This is because, as discussed above, the lower and upper functions needn't exist for $ \k \not\supseteq \k_0 $.
However, we will often encounter $ \k $ including the minimal information, which is technically not a valid input of the divergence.
To avoid this, we will use the notation $ \k_- $, defined as $ \k_- \deq \k \setminus \k_0 $, transforming $ \k $ to a valid input of the divergence.

The properties of the divergence differ based on the choice of the function $ \ell $.
For example, if we choose $ \ell $ to be the $ L_k $ norm (which we will denote by $ \lp k $), then the divergence will be \[
\divg {\lp k} \k \deq \sqrt[k]{\sum_{S \in \pot N}^{} \absolute{ \fce{\uf f}{S} - \fce{\lf f}{S}}^k}.
\]
This is exactly the $ L_k $-lenght of the longest diagonal in the hyperrectangle defined by the values of the upper and lower functions,
when viewed as $ 2^n $-dimensional vectors.\david{tohle by asi chtělo ještě rozepsat a říct jak tenhle rectangle nemusí obsahovat jen extenze atd.}

\subsection{Utopian Gap}

The divergence is a generalization of the so-called \emph{utopian gap} we have introduced in \cite{uradnik2024reducing}.
It has an interpretation in cooperative game theory, where the upper and lower functions are bounds on the characteristic function $ v $ of a cooperative game.
There, we want to split the value of $ \fce{v}{N} $ fairly among the players, using the Shapley value.
However, the Shapley value requires all the values of $ v $ to be known.

One way to estimate the Shapley value is to choose a function from the set of all superadditive extensions of $ v $ (we assume it is known that $ v $ is superadditive).
However, then each player has an incentive to choose the function which gives him the highest payoff, and without knowing the true value of $ v $,
there is no way to tell which choice is the most reasonable.
Hence the utopian gap defined to measure the difference between the players' choices, and the real value of $ v $.

\begin{defi}[Utopian gap]
  \label{def:gap}
  Let $ \left( N, \k, v \right) $ be an incomplete cooperative game.
  The \emph{utopian gap} of a cooperative game $ v $ is \[
    \dist v \deq \sum_{i \in N} \shapley[i] {v_i} - \fce{v}{N},
  \]
  where $ v_i $ is the \emph{utopian game} of player $ i $, defined as \[
    \fce{v_i}{S} \deq \begin{cases}
      \fce{\uf v}{S} & \text{if } i \in S, \\
      \fce{\lf v}{S} & \text{if } i \notin S.
    \end{cases}
  \]
\end{defi}

\section{Principal's Problems}

We will now introduce the main framework we will be working with, both in the theoretical and the experimental parts of this thesis.
The setup is as follows: we have a real-world situation modelled by a superadditive set function $ f: \pot N \to \R $ from a distribution of functions $ \fdist $, from which we only know $ \k_0 \subseteq \k \subseteq \pot N $ values.
There is a principal, who wants to know the rest of the values of $ f $ as precisely as possible.
However, she only has the budget to reveal $ \tau $ additional values.
She wants to choose $ \tau $ subsets from $ \pot N \setminus \k $ whose values to reveal (adding them to $ \k $), such that the bounds on the unknown values are as tight as possible, as measured by the divergence with a chosen function $ \ell $.

From this basic setup, we have two possible approaches to minimizing the divergence.
First, we can try to establish, which $ \tau $ subsets lead to the smallest divergence in expectation.
This leads to the definition of the \emph{offline principal's problem}.

\begin{defi}[Offline principal's problem] \label{def:offline-pp}
  Let $ \k_0 \subseteq \k \subseteq \pot N $ be the set of known subsets.
  Let $ \fdist $ be a given distribution of superadditive set functions, which all have the same values on $ \k $.
  Finally, let $ \ell: \funcs \times \funcs \to \R^+_0 $ be a divergence function.
  The \emph{offline principal's problem} is to find a set $ \s \subseteq \pot N \setminus \k $, such that \[
    \s = \argmin_{\substack{\abs{\s} = \tau \\ \s \subseteq \pot N \setminus \k }} \E_{f \sim \fdist} \left[ \divg \ell {\k_- \cup \s} \right].
  \]
\end{defi}

This is an approach which, if it were solved exactly, would give a \textquote{recipe} for the principal to follow, regardless of the already revealed values of $ f $.
However, there is an approach which leads to a potentially lower value of the divergence.
The flaw of the offline problem is that the optimal $ \s $ might depend on the specific instance $ f $ taken from $ \fdist $.
% \david{Observe that the principal doesn't use the values on the coalitions she encounters during the revelation process. These can be used to restrict the distribution $\fdist$, improving performance.}
Now, granted, the principal doesn't know all the values of $ f $.
However, in addition to knowing the values of $ \k $, she has the budget to reveal $ \tau $ extra values, which give her further information about the specific instance of $ f $ she is working with.
It might then be benefitial to spend some of the budget on exploring other values of $ f $,
and then use the remainder of the budget to reveal the values which are most beneficial, taking into account the values she has already revealed.
We call this approach the \emph{online principal's problem}, as the principal utilizes the online information she has gathered about $ f $ when choosing the next values to reveal.

The formal definition of the online principal's problem is a bit more involved.
Unlike the offline problem, here the solution will be a \emph{policy} $ \pi $, according to which the principal will choose the next values to reveal, based on the values she has already seen.
This input of the policy are the known subsets $ \k $, their corresponding values of $ f $, and the remaining budget, or the number of steps left, $ \tau $

\begin{defi}[Online policy valuation]\label{def:online-pv}
  Let $ \pi: \pot {\pot N \times \R} \times \N \to \dist {\pot N \setminus \k_0} $ be a policy.
  Let $ \left( f, \k \right) $ be an incomplete set function with minimal information, let $ \ell: \funcs \times \funcs \to \R^+_0 $ be a divergence function,
  and let $ \tau \in \N $ be the remaining budget, such that $ \tau \leq \absolute{\pot N \setminus \k} $.
  The \emph{policy valuation} of $ \pi $ is defined recursively as \[
    \val \tau \ell f \k \pi \deq \begin{cases}
      \E_{S \sim \pi(f\left< \k \right>, \tau)} \left[ \val {\tau - 1} \ell f {\k \cup S} \pi \right] & \text{if } \tau > 0, \\
      \divg \ell {\k_-} & \text{if } \tau = 0,
    \end{cases}
  \]
  where $ f \!\left< \k \right> \deq \left\{ \left( S, \fce{f}{S} \right) \suchthat S \in \k \right\} $.
\end{defi}

We can see that the policy valuation exactly quantifies the performance we have discussed above.
It measures the divergence after the budget has been spent, allowing the principal to make moves that might not be optimal in a \textquote{greedy} sense, meaning that they might not lead to the smallest divergence in the next step, but they might lead to a smaller final divergence.

Notice that the policy used here matches the definition of a policy we have given as \Cref{def:policy} in \Cref{sec:rl}, with $ \pot {N \times \R} \times \N $ as the state space, and $ \pot N \setminus \k_0 $ as the action space.
Having $ \tau $ as a part of the state is crutial for the policy, as (intuitively) it gives the information whether there is still budget to further explore the instance of $ f $, or if it is to instead focus on revealing the values which are most benefitial, given the values it has already seen.

\begin{defi}[Online principal's problem] \label{def:online-pp}
  Let $ \k_0 \subseteq \k \subseteq \pot N $ be the set of known subsets.
  Let $ \fdist $ be a given distribution of superadditive set functions, which all have the same values on $ \k $.
  Finally, let $ \ell: \funcs \times \funcs \to \R^+_0 $ be a divergence function.
  The \emph{online principal's problem} is to find a policy $ \pi: \pot {\pot N \times \R} \times \N \to \dist {\pot N \setminus \k_0} $,
  which receives \begin{enumerate}[ ]
    \item the known values $ \k $, along with the revealed values $ \fce{f}{S} $ for $ S \in \k $, and
    \item $ \tau $, the remaining number of steps,
  \end{enumerate}
  and outputs a probability distribution over $ \pot N \setminus \k $, such that $ \E_{f \sim \fdist} \val \tau \ell f \k \pi $ is minimal.
\end{defi}
