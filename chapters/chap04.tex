\label{chap:results}

% section intro
In this chapter, we show the experimental results of the above discussed algorithms.
We compare them with a baseline called \algRand{}, where the principal chooses which subsets to reveal uniformly at random from the as of yet unrevealed subsets.

% experimenty jsou i z clankuu
We first show results utilizing the game-theoretic motivation, with the Utopian Gap (\Cref{def:gap}) as the divergence.
These results are already published by \cite{uradnik2024reducing}, along with a very well-performing heuristic for uniform supermodular games.
In this thesis, we extend these results for different divergence functions, utilizing the $ L_k $-norms.
We compare these results to those with the Utopian Gap.
Finally, we show some examples documenting properties of the divergence already discussed above.

\section{Factory Game}

% factory game class
The first setting where we run our experiments is a class of cooperative games we call the \emph{factory games}.
\filip{citace?}
\david{ukradli jsme to z wiki, takže bych to asi necitoval :D https://en.wikipedia.org/wiki/Shapley_value}

\begin{defi}[Factory game]
  Let $ N = \left\{ 1, \ldots, n \right\} $ be a set of players and $ o \in N $ be an \emph{owner}.
  Then a \emph{factory game} has a characteristic function $ v_o $ defined as \[
    \fce{v_o}{S} \deq \begin{cases}
      \absolute{S} - 1 & \text{if $ o \in S $,} \\
      0                & \text{otherwise.}
    \end{cases}
  \]
\end{defi}

We imagine that this is a very simple model of a factory, with $ n-1 $ equally productive workers, and one owner of the factory.
If the owner is present in the coalition, the workers are allowed to work in the factory, each producing a value of $ 1 $.
Otherwise, the workers cannot work, and thus produce nothing.

The point of this example is not to precisely model a real-world situation.
It is rather to have a simple class of finite games with very simple structure, which are superadditive, and on which we can thus run our algorithms and see what they do.

\section{Largest Coalition Heuristic}

\label{ssec:largest}

% explain motivation
In the previous sections, we have seen that our methods work remarkably well on the uniform supermodular distribution for $ n=5 $.
Furthermore, all the algorithms end up choosing the same actions: revealing all the subsets\footnote{Or coalitions, but I will stick to the more general set function terminology in this section.} of size $ n-1 $.
After those are revealed, the Divergence decreases by about 99 \% (depending on the choice of $ \ell $).
This leads us to propose the heuristic we call the \algLC{}, which, for a budget of $ \tau = 2n + 2 $, reveals the subsets $ \k = \k_0 \cup \left\{ N \setminus \left\{ i \right\} \suchthat i \in N \right\} $.
