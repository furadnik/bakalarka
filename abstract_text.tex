\def\Abstract{
	Set functions offer a way to express the relationship between subsets of a finite ground set.
	This is used in countless fields, e.g., for modeling the utility of a player based on their received items in combinatorial auctions, or for modeling the value of coalitions in cooperative game theory.
	However, in applying set functions to a real-world problem, there is a significant roadblock: their size grows exponentially in size of the ground set, while finding out the value of even a single subset might be hard -- costing, e.g., money, time, or resources.
	In this thesis, we investigate an approach for dealing with this problem, where we specify a limit of how many subsets we are willing to reveal, and we put bounds on the remaining subsets.
	We aim to find out which subsets to reveal, such that the resulting ambiguity in the unknown subsets is minimized.
	We frame this as an optimization problem, for which we find both exact solutions, as well as approximations via reinforcement learning.
}
